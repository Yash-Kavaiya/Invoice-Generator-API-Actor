# Changelog

All notable changes to the Invoice Generator API Actor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added
- Initial release of Invoice Generator API Actor
- Support for 4 invoice templates (Modern, Classic, Minimal, Professional)
- Multi-format output generation (PDF, HTML, JSON)
- Support for 9 major currencies (USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY, INR)
- Automatic invoice calculations (subtotals, taxes, discounts, totals)
- Automatic invoice numbering
- Due date calculation
- Custom fields support
- Company logo integration
- Payment terms and banking information
- Purchase order tracking
- Tax ID/VAT number support
- Comprehensive input validation
- Example input files for different use cases
- Detailed documentation and README

### Features
- Handlebars-based templating system
- Puppeteer PDF generation
- Apify SDK integration
- Key-value store for invoice file storage
- Dataset for invoice metadata tracking

### Utilities
- Currency formatting utilities
- Date handling and formatting
- Invoice number generation
- Calculation engine for amounts
- Input validation system
